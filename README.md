# heartbeat
Heartbeat Binaries

## Install

Run the following command to download and install the correct binary for your platform:

```bash
curl -fsSL https://raw.githubusercontent.com/NIZU-io/heartbeat/main/install.sh | bash
```

Or with `wget`:

```bash
wget -qO- https://raw.githubusercontent.com/NIZU-io/heartbeat/main/install.sh | bash
```

This will detect your architecture, download the appropriate binary, and install it to `/usr/local/bin`. You may need to run with `sudo` if you don't have write access to that directory:

```bash
curl -fsSL https://raw.githubusercontent.com/NIZU-io/heartbeat/main/install.sh | sudo bash
```
